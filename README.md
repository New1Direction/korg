## Overview
Korg/Yvaeh Mode is an advanced, adversarial multi‑agent orchestration harness that turns a single CLI prompt into a self‑compounding swarm of intelligent agents. It features:
- **Four persona agents** (Captain, Harper, Benjamin, Lucas) that run in parallel, each with a distinct strategy.
- **Live contract negotiation** powered by BERT similarity scoring.
- **Dynamic scaling** up to 16 agents based on telemetry.
- **Persistent blackboard & `.ktrans` crash‑recovery** for fault‑tolerant execution.
- **Automatic post‑run reconcile & synthesize** that enriches the knowledge vault.

## Quick Start
```bash
# Install via Homebrew (macOS)
brew install korg

# Or run the Docker image
docker run -it ghcr.io/clubpenguin/korg:latest "Refactor the authentication module"

# Run a campaign directly
korg "Refactor the authentication module to support automatic retries and rate limiting"
```
